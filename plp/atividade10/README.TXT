Instruções de uso:

1 - Tenha o Python instalado no seu computador com a versão mínima sendo 3.10(devido ao uso do Match-case ele não funcionará com outras versões).

2 - Extraia os arquivos "dequeue.py", "queue.py" e  "stack.py" que se encontram na pasta arquivos desse arquivo compactado para um diretório de sua escolha,
mantendo as 3 sempre no mesmo diretório.

3 - para usar um dos arquivos Stack ou Queue, abra o terminal de seu computador, e entre no diretório em que os arquivos se encontram,
em seguida execute o seguinte comando no terminal windows: python "nome do arquivo desejado".py e siga as intruções que aparecerão no teminal após isso.
